# Math task 2 

In the second task you are already provided with <strong>two calculations</strong>, with the results stored in the <strong>variables</strong> <em>result</em> and <em>result2.</em><br>
But these results <em>aren't what we want</em>; you'll need to take the <strong>calculations</strong> and change them to <em>give us what we want.</em>

<em>What do we want?</em> After <em>multiplying</em> the two results together and <em>formatting</em> the result to 2 decimal places, the final result should be 10.42.

Try <em>updating</em> <strong>the starting point</strong> to <em>recreate</em> the finished example, following these steps:

1. <em>Write</em> a calculation that <em>multiplies</em> result and result2 together and assigns the result back to <em>result.</em> This will require assignment shorthand.

2. <em>Write</em> a line of code that takes result and <em>formats</em> it to 2 decimal places, storing the result of this in a <strong>variable</strong> called <em>finalResult.</em>

3. <em>Check</em> the <strong>data type</strong> of <em>finalResult</em> using <strong>typeof;</strong> you'll probably see that it is actually of <strong>string</strong> type! <em>Write</em> a line of code that converts it to a <em>number</em> type, storing the result in a <strong>variable</strong> called <em>finalNumber.</em>

4. The <strong>value</strong> of <em>finalNumber</em> needs to be <em>10.42.</em> Go back and <em>update</em> the original calculations you were provided with so that they give this final result. Don't update the <strong>numbers</strong> or the <strong>operators.</strong>
